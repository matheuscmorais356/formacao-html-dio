Estrutura básica do HTML

(Voltar)

A estrutura básica do HTML organiza e define o conteúdo de uma página web de forma hierárquica. No topo, temos o <!DOCTYPE html>, que informa ao navegador que o documento é HTML5. Em seguida, o elemento <html> envolve todo o conteúdo e usa o atributo lang para especificar o idioma da página.

Dentro da tag <html>, a estrutura é dividida em dua partes principais:

  1. Cabeçalho (<head>): Contém informações sobre a página que não são exibidas diretamente no corpo de página, como o título, ícones, links para estilos (CSS) e metadados importantes, como o <meta charset="UTF-8">, que define a codificação de caracteres.

  2. Corpo (<body>): Inclui todo o conteúdo visível da página, como textos, imagens, links, tabelas e formulários. Os elementos do <body> são o que o usuário verá e interagirá diretamente. 

    
      <!DOCTYPE html>
      <html>
        <head>

        </head>
        <body>

        </body>
      </html>

O que são tags

(Voltar)

As tags HTML são os elementos básicos que estruturam e organizam o conteúdo de uma página web. Cada tag tem uma função específica e é usada para definir diferentes tipos de conteúdo e elementos de formatação. Tags HTML são representadas por palavras-chave colocadas entre colchetes angulares, como <p>, <h1>, ou <div>. Elas normalmente vêm em pares: uma tag de abertura, como <p>, e uma de fechamento, como </p>, envolvendo o conteúdo que elas definem.

As tags são fundamentais para criar a hierarquia e semântica de uma página. Por exemplo:

  • Tags de título (<h1> a <h6>): Definem títulos e subtítulos, com <h1> sendo o mais importante e <h6> o menos importante.
  • Tag de parágrafo (<p>): Define blocos de texto.
  • Tags de lista (<ul>, <ol>, <li>): Criam listas ordenadas ou não ordenadas e seus itens.
  • Tags de link (<a>): Criam links para outras páginas ou seções.
  • Tags de imagem (<img>): Inserem imagens na página.

Além disso, há tags mais complexas, como <table> para tabelas, <form> para formulários e <div> e <span>, que são usadas para organizar e estilizar grupos de conteúdo.

Cada tag tem seu papel semântico, ajudando navegadores, motores de busca e tecnologias assistivas a interpretar e exibir o conteúdo corretamente.

O que são atributos

(Voltar)

Atributos HTML são informações adicionais inseridas dentro das tags para modificar ou definir propriedades dos elementos. Eles ajudam a configurar o comportamento, a aparência ou as características de um elemento na página. Normalmente, os atributos são escritos dentro da tag de abertura e seguem o formato nome="valor", por exemplo: 

    
      <a href="https://exemplo.com">Clique aqui</a>

Nesse exemplo, href é um atributo que define o destino do link.

Atributos globais

Os atributos globais são aqueles que podem ser utilizados em qualque tag HTML.

  • id: Identifica unicamente o elemento na página.
  • class: Atribui uma ou mais classes CSS ao elemento para estilização.
  • style: Define estilos CSS diretamente no elemento.
  • title: Define um texto que aparece como dica quando o usuário passa o mouse sobre o elemento.

Atributos Específicos

São próprios de determinadas tags e configuram funções exclusivas desses elementos.

  • Em links (<a>):
    • href: define o destino do link
    • target: determina como abrir o link.
  • Em imagens (<img>):
    • src: define a URL da imagem
    • alt: fornece uma descrição alternativa.
  • Em inputs de formulário (<input>):
    • type: define o tipo de entrada
    • placeholder: mostra um texto sugestivo
    • value: define o valor padrão.

Tags de texto

(Voltar)

Em HTML, existem várias tags que ajudam a formatar e organizar textos. Cada tag tem uma função específica que contribui para a estrutura e semântica da página, facilitando a leitura e o estilo do conteúdo. Abaixo estão algumas das principais tags de texto:

  • <p>: Define um parágrafo de texto. É uma das tags mais comuns e cria uma separação entre blocos de texto.
  • <h1> a <h6>: Representam títulos e subtítulos, com <h1> sendo o título principal e <h6> o menos importante. Esses níveis de título ajudam a organizar o conteúdo e são muito importantes para SEO.
  • <strong>: Aplica ênfase forte ao texto, exibindo-o em negrito. É usado quando queremos dar destaque semântico ao conteúdo, não apenas visual.
  • <em>: Marca o texto com ênfase, normalmente exibindo-o em itálico. Diferente do <i>, o <em> possui significado semântico, indicando um trecho importante para o contexto.
  • <span>: Um elemento genérico para estilizar partes do texto. Diferente do <div>, que é um contêiner em bloco, o <span> é um contêiner em linha, ideal para aplicar estilos específicos em pequenas partes do texto.
  • <br>: Insere uma quebra de linha no texto. É útil para criar uma nova linha sem começar um novo parágrafo.
  • <blockquote>: Usado para inserir uma citação longa ou bloco de citação. Normalmente, o navegador aplica uma indentação, dando destaque à citação.
  • <code>: Exibe texto em estilo de código, com uma fonte monoespaçada, ideal para exibir trechos de código ou comandos técnicos.
  • <mark>: Realça o texto como se estivesse marcado, destacando-o visualmente como relevante ou importante no contexto.
  • <del>: Indica texto deletado ou removido, geralmente aparecendo com uma linha no meio do texto.
  • <ins>: Indica texto inserido ou adicionado recentemente, geralmente exibido com um sublinhado.

Essas tags ajudam a criar uma estrutura de texto mais rica e semântica em HTML, facilitando a compreensão e acessibilidade do conteúdo para usuários e motores de busca.

Listas

(Voltar)

Em HTML, as listas ordenadas e não ordenadas são usadas para organizar informações em formato de lista. Elas são criadas com as tags <ol>, <ul>, e <li>, e cada uma tem uma finalidade e uma forma de apresentação específicas.

Listas ordenadas <ol>

Uma lista ordenada é usada quando a ordem dos itens importa. Os itens da lista são numerados automaticamente pelo navegador. Para criar uma lista ordenada, usamos a tag <ol> (Ordered List), e cada item da lista é marcado com a tag <li> (List Item). 

    
      <ol>
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
      </ol>

Exemplo de lista ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas não ordenadas <ul>

Uma lista não ordenada é usada quando a ordem dos itens não é importante. Os itens da lista são marcados com pontos (bullets) ou outros símbolos. Para criar uma lista não ordenada, usamos a tag <ul> (Unordered List), e cada item da lista também é marcado com a tag <li> (List Item). 

    
      <ul>
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
      </ul>

Exemplo de lista não ordenada:

  • Primeiro item
  • Segundo item
  • Terceiro item

Links

(Voltar)

Em HTML, links são elementos que permitem a navegação entre páginas, eles são criados com a tag <a>, que significa “anchor” (âncora). Os links podem apontar para URLs, navegar entre páginas da mesma aplicação, navegar para partes específicas da página ou mesmo disparar downloads.

Criando um Link Básico

A estrutura básica de um link é com a tag <a> e o atributo href (Hypertext REFerence), que define o destino do link. 

    
      <a href="https://www.exemplo.com">Visite o Exemplo</a>

Atributos Comuns do Link

href

Define o destino do link. Pode ser uma URL completa, uma URL relativa, um identificador de âncora ou o caminho de um arquivo local.

Exemplo:

    
      <a href="https://www.exemplo.com">Abrir em uma nova aba</a>

target

Define onde o link será aberto.

  • _self: (Padrão) abre o link na mesma aba.
  • _blank: abre o link em uma nova aba ou janela.
  • _parent: abre o link na janela ou quadro pai.
  • _top: abre o link na janela ou quadro superior.

Exemplo:

    
      <a href="https://www.exemplo.com" target="_blank">Abrir em uma nova aba</a>

rel

Define a relação entre o documento atual e o link. É especialmente útil quando target="_blank" é usado, para melhorar a segurança.

  • rel="noopener": Impede que a página de abertura obtenha qualquer tipo de acesso à página original.
  • rel="noreferrer": Instrui o navegador a omitir o cabeçalho “Referer” e não vazar informações sobre o referenciador.

Exemplo:

    
      <a 
        href="https://www.exemplo.com" 
        target="_blank" 
        rel="noopener noreferrer"
      >
        Link Seguro
      </a>

title

Exibe um texto ao passar o mouse sobre o link. Isso pode melhorar a acessibilidade e fornecer mais informações ao usuário.

Exemplo:

    
      <a href="https://www.exemplo.com" title="Visite o site Exemplo">Exemplo</a>

Links para Partes Específicas do Documento

É possível criar links para seções específicas da mesma página usando identificadores (IDs).

  1. Atribua um id ao elemento de destino. 
    2.       
        <h2 id="secao">Seção Específica</h2>
  2. Crie o link usando # seguido pelo valor do id. 
    3.       
        <a href="#secao">Ir para a Seção</a>
